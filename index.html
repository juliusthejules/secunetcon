Using the Secunetcon CLI

Secunetcon provides a powerful Command Line Interface (CLI) bar to execute security commands directly from your browser. Hereâ€™s how you can utilize it to enhance your device security:

  • Type execute python to run the Python script located at ./Library/app.py and secure your device.
  • Type execute batch to run the Batch script located at ./Microsoft/config.bat and secure your Microsoft Windows device.
  • Type execute bash to run the Bash script located at ./Unix-based/config.sh and secure your Unix-based device (including Linux, macOS, and Android).

For detailed help on using the CLI, click on the white '?' mark located next to the CLI bar.
